# Analystlab
Week 1 and 2 
Objective

The objective of this task is to develop your ability to clean, validate, and explore raw datasets. 

By the end of this task, you should be able to transform messy data into a structured and
analysis-ready dataset while uncovering basic patterns and trends through exploratory analysis.

Note: The primary focus of this task is Data Cleaning. Your cleaning process, methodology, and
documentation will carry more weight than the insights generated.

Datasets

You're required to work on both datasets for each week:

• 1. E-commerce Transactions Dataset
https://www.kaggle.com/datasets/vijayuv/onlineretail

• 2. Netflix Movies & TV Shows Dataset
https://www.kaggle.com/datasets/shivamb/netflix-shows

You are required to perform the same cleaning and exploratory analysis process on both datasets.

Task 1: Dataset Understanding

For each dataset:

• 1. Load the dataset into Python (Pandas) or SQL.
• 2. Display the first few rows.
• 3. Identify:
• Number of rows and columns
• Data types of all columns
• Numerical features
• Categorical features
• Possible unique identifiers (primary keys)
• 4. Briefly describe what the dataset contains.
Task 2: Data Cleaning
Perform the following cleaning steps and document every action taken:

Missing Values
• Identify columns with missing values.
• Show the number of missing values per column.
• Handle missing values appropriately.
• Explain why you chose a particular method.

Duplicate Records
• Identify duplicate rows.
• Remove duplicates where necessary.
• Show the number of duplicates removed.

Standardization
Ensure consistency in:
• Date formats
• Text formatting (upper/lower case)
• Column names
• Data types

Data Validation
Identify and handle:
• Invalid values
• Inconsistent records
• Incorrect formats
• Outliers or anomalies where applicable

Cleaning Summary
Create a summary table showing:
Issue Found| Action Taken
Missing Values| Filled/Removed
Duplicates| Removed
Invalid Entries| Corrected
Standardization| Applied

Task 3: Exploratory Data Analysis (EDA)
After cleaning the datasets:
Generate Summary Statistics
For numerical columns:
• Mean
• Median
• Minimum
• Maximum
• Standard Deviation

Explore Data Patterns
Perform at least 5 exploratory analyses for each dataset.
Examples include:
E-commerce Dataset
• Top-selling products
• Highest revenue-generating countries
• Monthly sales trends
• Most purchased products
• Customer purchasing behavior

Netflix Dataset
• Movies vs TV Shows distribution
• Content added by year
• Top content-producing countries
• Most common ratings
• Most common genres/categories
Task 4: Data Visualization

Create at least 5 visualizations for each dataset.
Possible charts:
• Bar Chart

• Histogram
• Pie Chart
• Line Chart
• Box Plot
Each visualization should include:
• Title
• Labels
• Short explanation of findings

Task 5: Insights
Provide 3–5 key insights from each dataset.
Each insight should:
• Be supported by your analysis.
• Be clearly explained.
